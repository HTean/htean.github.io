OnlyRainsOnWeekends — Weather Pattern Ranking Engine

A data-driven platform that ranks cities by their tendency to rain specifically on weekends — built from curiosity, powered by historical weather APIs, and surprisingly useful for trip planning.

The Problem

Ever notice that it seems to rain more on weekends than weekdays? That nagging feeling — sometimes called the "weekend effect" — has been studied in atmospheric science and is partially real: human activity during the week can shift precipitation patterns toward weekend peaks in some regions.

I wanted to quantify this properly. Which cities actually have measurably worse weekend weather? Which have better? And can you build a tool that makes this useful for travelers and remote workers choosing where to spend their time?

What I Built

OnlyRainsOnWeekends is a web platform that pulls historical daily precipitation data from the Open-Meteo API for 140+ cities worldwide, then calculates a "weekend bias score" — comparing average weekend rainfall against weekday rainfall over a multi-year window.

Cities are ranked and filterable by region, climate type, and season. The result is an interactive, data-driven reference for anyone planning where to work, travel, or live.

Technical Approach

The data pipeline fetches historical daily precipitation records for each city going back 5+ years, then classifies each day as weekend or weekday. A bias score is computed as the percentage difference between mean weekend and mean weekday precipitation, corrected for seasonal variance.

The frontend is intentionally lightweight — pure HTML, CSS, and vanilla JavaScript with no heavy frameworks, keeping load times fast for a globally accessed site. City data is pre-processed and cached as JSON so the UI is instant.

Reflections

What started as a weekend curiosity project turned into a genuinely useful tool. The biggest technical challenge was normalizing rainfall patterns across vastly different climates — a 2mm difference means nothing in a tropical city but is significant in a desert region. Applying climate-relative scoring was the key insight that made the rankings meaningful.

Next steps: adding seasonal views and a "best month to visit" breakdown per city.
