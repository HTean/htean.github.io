My Navigator — Personal Web Dashboard

A minimalist personal dashboard for organizing and instantly accessing your most-used websites — built because bookmark folders are a mess and browser tabs are chaos.

The Problem

Power users — developers, analysts, researchers — tend to accumulate dozens of essential but disparate tools: internal dashboards, APIs, documentation pages, client portals, and reference sites. Bookmark managers are cluttered. New-tab extensions feel bloated. I wanted something clean, fast, and entirely mine.

What I Built

My Navigator is a personal browser dashboard — a custom new-tab replacement that lets you save, categorize, and search websites using a clean card-based layout. Links are stored persistently via Supabase so they sync across devices and survive browser resets.

The interface is deliberately minimal: no icons, no ads, no noise — just your links, organized the way you want them, accessible in one click.

Technical Approach

Built as a pure frontend HTML/CSS/JS app with Supabase as the backend database layer. Authentication is handled via Supabase's built-in auth so the dashboard is private. A lightweight tag system allows flexible categorization without a complex UI.

Reflections

This is one of those tools I use every single day. The constraint of "no frameworks, no build step" forced clean, readable code and made the page load near-instantly. I'd eventually like to add shared workspaces so teams can maintain a shared link library.
